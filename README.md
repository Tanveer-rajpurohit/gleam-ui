# Gleam UI

Motion-first React components for expressive interfaces.

Gleam UI is an open-source collection of animated and 3D React
components built with GSAP, Motion and React Three Fiber. Install
the package or explore the complete component source.

## Features

- Motion-first React components
- WebGL and React Three Fiber experiences
- GSAP-powered scroll interactions
- TypeScript-first component APIs
- Light and dark theme support
- Tree-shakeable exports
- Full source available in the documentation
